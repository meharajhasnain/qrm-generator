# QRM Generator

A precision QR code generator built with React and Vite.

## Features
- Direct text QR generation
- Multiple node styles (Square, Rounded, Smooth)
- Custom accent colors
- Logo embedding option
- High precision error correction toggle
- Download as PNG

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build for production (e.g. GitHub Pages):
   `npm run build`
